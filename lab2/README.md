## 4. Как реализовано (Теория и фрагменты кода)

### 4.1. Инициализация и событийно-ориентированная модель

**Теория:** В современном JavaScript для привязки логики к действиям пользователя применяется событийно-ориентированный подход (Event-Driven Programming). Использование устаревших inline-обработчиков (вида `onclick`) было заменено на метод `addEventListener`. Для предотвращения попыток JS-кода обратиться к еще не отрисованным элементам (Render Blocking), весь код инкапсулирован внутри слушателя события `DOMContentLoaded`, которое срабатывает сразу после построения DOM-дерева браузером.

**Фрагмент кода (Подписка на события):**

```javascript
document.addEventListener('DOMContentLoaded', () => {
    // Получение узлов DOM
    const digitButtons = document.querySelectorAll('[id ^= "btn_digit_"]');

    // Навешивание слушателей в цикле
    digitButtons.forEach(button => {
        button.addEventListener('click', function() {
            const digitValue = button.innerHTML;
            onDigitButtonClicked(digitValue);
        });
    });
});

```

### 4.2. Архитектурный паттерн «Стратегия» для вычислений

**Теория:** Использование классической конструкции `switch-case` для маршрутизации математических операций нарушает принцип открытости/закрытости (OCP из SOLID). Было принято решение использовать структуру данных «Словарь» (объектные мапы), что является реализацией паттерна Стратегия (Strategy). Операции разделены по арности: `binaryManeuvers` (требуют 2 числа и нажатия `=`) и `unaryManeuvers` (выполняются мгновенно).

**Фрагмент кода (Модели операций пульта SpaceX):**

```javascript
// Бинарные операции
const binaryManeuvers = {
    'btn_op_plus': (x, y) => x + y,
    'btn_op_mult': (x, y) => x * y,
    // ...
};

// Унарные операции (включая индивидуальную D2S и факториал с защитой)
const unaryManeuvers = {
    'btn_op_sqrt': x => Math.sqrt(x),
    'btn_op_factorial': x => factorial(x), // Генерирует throw new Error при x < 0
    'btn_op_d2s': x => x / 1.02749         // Earth Days to Mars Sols
};

```

Ошибки (например, деление на ноль или отрицательный факториал) перехватываются блоком `try...catch` и переводят пульт в режим блокировки `ifError = true`, отображая красное предупреждение.

### 4.3. Управление состоянием и аппаратная память

**Теория:** Приложение реализует концепцию сохранения локального состояния (State). Для работы кнопок серии `M` выделена отдельная переменная `memoryRegister`. Функция-хелпер `getCurrentScreenValue()` динамически определяет, какое именно число (первое введенное или второе) сейчас находится на экране, чтобы добавить его в память.

**Фрагмент кода (Логика регистра памяти):**

```javascript
let memoryRegister = 0;

document.getElementById('btn_mem_plus').addEventListener('click', () => {
    if (ifError) return;
    memoryRegister += getCurrentScreenValue();
});

document.getElementById('btn_mem_recall').addEventListener('click', () => {
    if (ifError) return;
    let memString = memoryRegister.toString();

    // Записываем значение из памяти в активный операнд
    if (!selectedOperation) {
        a = memString;
        displayNumber(a);
    } else {
        b = memString;
        displayNumber(b);
    }
});

```

### 4.4. Манипуляция DOM в многостраничной среде

**Теория:** В связи с дополнительным заданием по разбиению интерфейса на несколько HTML-страниц, логика смены темы была вынесена в независимый скрипт `change_theme.js` и подключена ко всем файлам. Смена темы происходит за счет манипуляции атрибутом `data-theme` корневого тега `<body>`. Переключение акцентного цвета экрана вычислений реализовано через безопасное API `classList.toggle`, что обеспечивает строгую изоляцию бизнес-логики от CSS-стилей.

**Фрагмент кода (Смена темы оформления):**

```javascript
document.getElementById('btn_change_theme').addEventListener('click', () => {
    const currentTheme = document.body.getAttribute('data-theme');
    const newTheme = currentTheme === 'dark' ? 'light' : 'dark';

    // Установка состояния на уровне body
    document.body.setAttribute('data-theme', newTheme);
    // Синхронизация выпадающего списка
    document.getElementById('select_theme').value = newTheme;
});

```
